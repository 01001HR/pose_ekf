# pose_ekf
Extented Kalman Filter for 6D pose estimation using gps, imu, magnetometer and sonar sensor.

#state for kalman filter
0-3 quaternion

4-6 Pn Pe Pd

7-9 Vn Ve Vd

10-12 bwx bwy bwz

13-15 bax bay baz
